# Hsing Stock AI

個人持股決策助理 PWA。

## 功能

- 首頁市場燈號
- 四檔持股追蹤
- 先行指標
- AI 策略
- 跌破關鍵價減碼提醒
- 漲太多減碼提醒
- 後續加碼提醒
- 手機加入主畫面

## GitHub Pages 部署

1. 到 GitHub 建立新 repository，例如 `hsing-stock-ai`
2. 將本資料夾內所有檔案上傳到 repository 根目錄
3. 到 repository 的 `Settings`
4. 進入 `Pages`
5. `Build and deployment` 選 `GitHub Actions`
6. 等待 Actions 完成
7. 網址會類似：

   `https://你的帳號.github.io/hsing-stock-ai/`

## 手機安裝

- iPhone Safari：分享按鈕 -> 加入主畫面
- Android Chrome：選單 -> 安裝應用程式

## 安全提醒

目前密碼鎖是前端 MVP 保護，不是正式登入系統。若要長期公開使用，建議加入後端登入或 Cloudflare Access。
